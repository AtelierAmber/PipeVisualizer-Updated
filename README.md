# Pipe Visualizer

Visualize the layout and fluids in Factorio pipe networks.

## Installation

Download on the official Factorio [mod portal](https://mods.factorio.com/mod/PipeVisualizer).

## Features

Draw individual systems with `H`, or draw everything on screen with `Shift + H`. Press `Control + H` to toggle between color-by-fluid and color-by-system.
